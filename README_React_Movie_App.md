# 🎬 React Movie App | Search & Favorites

I don’t usually watch movies… but I built an app for them 😄

This project is a React-based Movie Application that allows users to search for movies and manage their favorites. It was built as part of my hands-on learning to understand how real-world React applications work.

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1. Clone the repository
```bash
git clone https://github.com/Hariprasath191/React-Movie.git
```

### 2. Navigate to project folder
```bash
cd React-Movie
```

### 3. Install dependencies
```bash
npm install
```

### 4. Run the development server
```bash
npm run dev
```
